Parallel

Namespace: Microsoft.FSharp.Collections
Parent: Array

Provides parallel operations on arrays


Values and Functions
Name Description
choose

Signature: chooser:('T -> 'U option) -> array:'T [] -> 'U []
Attributes: CompiledNameAttribute

Apply the given function to each element of the array. Return the array comprised of the results "x" for each element where the function returns Some(x).

Parameters

chooser

The function to generate options from the elements.

array

The input array.

Returns: 'U[]

Remarks

Performs the operation in parallel using System.Threading.Parallel.For. The order in which the given function is applied to elements of the input array is not specified.
