Printf

Namespace: Microsoft.FSharp.Core
Parent: Microsoft.FSharp.Core

Extensible printf-style formatting for numbers and other datatypes

Remarks

Format specifications are strings with "%" markers indicating format placeholders. Format placeholders consist of: where the type is interpreted as follows: The optional width is an integer indicating the minimal width of the result. For instance, %6d prints an integer, prefixing it with spaces to fill at least 6 characters. If width is '*', then an extra integer argument is taken to specify the corresponding width. Valid flags are: The printf '#' flag is invalid and a compile-time error will be reported if it is used.
