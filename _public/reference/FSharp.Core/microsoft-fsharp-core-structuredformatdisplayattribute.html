StructuredFormatDisplayAttribute

Namespace: Microsoft.FSharp.Core
Parent: Microsoft.FSharp.Core

This attribute is used to mark how a type is displayed by default when using '%A' printf formatting patterns and other two-dimensional text-based display layouts. In this version of F# valid values are of the form . The property names indicate properties to evaluate and to display instead of the object itself.
